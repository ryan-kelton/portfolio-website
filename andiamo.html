Andiamo al Café

Project Overview

Andiamo al Café is a small town coffee shop who was in need of a brand identity and online presence. After creating the logo and brand visual style I designed and developed their interactive website.

Link to Github repository Link to live site on Github

Scope

  1. Brand identity
  2. User interface design
  3. Interactive website features

Brand Identity

Explanation

Andiamo al Café's tagline is "The Heart And Soul of Your Daily Life". Taking this into consideration, I created a logo that I though encapsulated their mission. The logo is in the shape of a coffee bean with a heart symbol in the center.

After the brand identity was finished I created three coffee bag mockups in Photoshop for the product display section section on the website.

User Interface Design

For this project I started with a simple wire-frame to outline the hierarchy and layout of the site. I then created a quick mockup with Adobe Illustrator with the text and images I planned on using in the finished site.

Interactive Website Features

Product Display

When the user clicks on one of the coffee roast options, the corresponding coffee bag image and description replaces the prior content.

Contest Section

The user will input a number, and if it matches the randomly generated number in JavaScript than the user wins a free coffee.

Contact Form With Validation

It was critical that the contact form have error and valid states and a confirmation message.

Reflection

I was surprised to discover how much I enjoyed writing JavaScript. This was my first project using the programming language, and I didn't know what to expect.

I found it helpful to write out my objectives before I started writing any code. This made it easier to solve the problem in manageable steps.

Looking back I think there are more efficient ways to create a product display. Although I would have to do more research to be sure. I now know a better way to create a dark mode toggle, and if I could go back I would change the way I implemented it.
